注释

在CMand中，注释跟C++是一样的，但是如果你不了解C++也不用慌，因为注释非常简单

1. 单行注释

单行注释以//开头，注释内容直到行尾结束。

// 这是一个单行注释

2. 多行注释

多行注释以/*开头，以*/结尾，注释内容可以跨越多行。

/*
这是一个多行注释
*/

3. 易语言注释

如果你使用的是简易IDE的话，那么可以直接使用易语言的注释方式。

‘ 这是一个注释

单引号后面就是注释。
